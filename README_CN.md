# Dota 2 AI 助手

一个功能完整的 Dota 2 AI 助手，支持全部 123 个英雄！

## ✨ 特性

- 🎮 **实时游戏监控** - 通过 GSI 获取游戏状态
- 🧠 **智能决策** - LLM 战略 + 规则战术
- 🦸 **全英雄支持** - 123 个英雄的完整数据
- 📊 **可视化界面** - 清晰的终端显示
- 🎯 **补刀算法** - 精确的补刀时机计算
- 🤖 **自动控制** - 可选的输入模拟（有风险）

## 🚀 快速开始

\`\`\`bash
cd /Users/yuri/dota2bot/dota2_assistant

# 安装
./install.sh

# 运行（观察模式）
python3 main.py

# 启动 Dota 2 并进入游戏
\`\`\`
